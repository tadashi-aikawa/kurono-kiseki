## channelルール

### まとめ

- 一番早くに最初の章をクリアした人が
	- [ ] channelをつくる (序章か1章かはナンバリングに響く)
	- [ ] other_kiseki に↑を周知する
- 章をクリアしたら...
	- 自分が一番速ければ
		- [ ] 新しいchannelを作る (連番)
	- 1つ前のchannelで既に作成された報告があれば
		- [ ] その人に招待をお願いする
